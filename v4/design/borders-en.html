Description

Borders can be added to grids or other elements in order to separate or call attention to a section of content.

Border options

There are 4 borders that can be added:

  • .brdr-tp
  • .brdr-rght
  • .brdr-bttm
  • .brdr-lft

Adding borders to a grid cell

Borders can be added to a grid cell. Since grids have a left and right gutter, the border will be appear on outside edge of the gutter.

  • Top border

    <div class="col-md-3 brdr-tp">

  • Right border

    <div class="col-md-3 brdr-rght">

  • Top bottom

    <div class="col-md-3 brdr-bttm">

  • Left border

    <div class="col-md-3 brdr-lft">

Adding borders to content

  • Top border
    <p class="brdr-tp">

  • Right border
    <p class="brdr-rght">

  • Bottom border
    <p class="brdr-bttm">

  • Left border
    <p class="brdr-lft">

Adding multiple borders to the same element

  • Top and left border
    <div class="col-md-3 brdr-tp brdr-lft">

  • Bottom and right border
    <div class="col-md-3 brdr-bttm brdr-rght">

  • Top, right, bottom and left border
    <p class="brdr-tp brdr-rght brdr-bttm brdr-lft">

Remove border radius

Apply brdr-rds-0 to remove the border radius that appears on certain design elements,

  • Default well

    <div class="well">
	...
</div>

  • Well with border radius removed

    <div class="well brdr-rds-0 ">
	...
</div>
