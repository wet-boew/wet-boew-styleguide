Default pagination

Simple pagination inspired by Rdio, great for apps and search results. The large block is hard to miss, easily scalable, and provides large click areas.

  • Previous
  • 1
  • 2
  • 3
  • 4
  • 5
  • Next
<ul class="pagination">
	<li><a href="#" rel="prev">Previous</a></li>
	<li><a href="#">1</a></li>
	<li><a href="#">2</a></li>
	<li><a href="#">3</a></li>
	<li><a href="#">4</a></li>
	<li><a href="#">5</a></li>
	<li><a href="#" rel="next">Next</a></li>
</ul>

Disabled and active states

Links are customizable for different circumstances. Use .disabled for unclickable (hidden) links and .active to indicate the current page.

  • Previous
  • 1 (current)
  • 2
  • 3
  • 4
  • 5
  • Next
<ul class="pagination">
	<li class="disabled"><a href="#" rel="prev">Previous</a></li>
	<li class="active"><a href="#">1 <span class="wb-inv">(current)</span></a></li>
	...
</ul>

Simple pager

Quick previous and next links for simple pagination implementations with light markup and styles. It's great for simple sites like blogs or magazines.

Default example

By default, the pager centers links.

  • Previous
  • Next
<ul class="pager">
	<li><a href="#" rel="prev">Previous</a></li>
	<li><a href="#" rel="next">Next</a></li>
</ul>

Aligned links

Alternatively, you can align each link to the sides:

  • Previous
  • Next
<ul class="pager">
	<li class="previous"><a href="#" rel="prev">Previous</a></li>
	<li class="next"><a href="#" rel="next">Next</a></li>
</ul>
