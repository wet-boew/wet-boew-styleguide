Description

Too icon sets are, Glyphicons and Font Awesome, are featured below. The icons are glyphs, which means that they are picture-based fonts, similar to wingdings. However, unlike a traditional font, they serve ony decorative purposes. Therefore, real text must accompany the icon to ensure accessibility requirements are met. That text can be visible or invisible, but must be present.

Glyphicons

200 glyphs in font format from the Glyphicon Halflings set. Glyphicons Halflings are normally not available for free, but their creator has made them available for Bootstrap free of cost. This has been added to the core build of WET.

Font Awesome

400+ glyphs in font format from the Font Awesome set. Font Awesome is a free resource. Due to the increased number of options balanced with the desire to reduce the file size of a page, these icons can be added to page. They are not part of the core build.

Instructions

Simply add this code within the <head> of the page: <link href="//netdna.bootstrapcdn.com/font-awesome/4.0.3/css/font-awesome.css" rel="stylesheet">

If an icon is needed, attempt to use the glyphicon option as these icons are already being downloaded by the user. However if more options are needed, bring in the Font Awesome set as needed.

Third party icons

Using icons to represent a third party's logo can be contrary to that company's own brand asset policy. For instance:

  • Facebook's Do's and Don'ts states: "Use any icons, images or trademarks to represent Facebook other than what is found on this resource center"
  • Twitter's brand assets and guidelines states: "Our logo is always either blue or white. The Twitter bird is never shown in black or other colors.

In addition, pixel-perfect spacing and other considerations are required. So it is best, legally-speaking, to use the actual image provided by the third party themselves. While this is not as convenient, it is the correct approach.

Correct use

Although icons add a decorative component to the page, they do not enhance usability.

Use when

Use icons consistently throughout the site, and use icons that look like their actions.

Do not use when

Do not only use icons to convey information. Ensure you include appropriate text with them to provide context and meaning.
