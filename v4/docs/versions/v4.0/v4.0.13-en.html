What's New

  • Multimedia: Fixed the controls not loading for the Flash fallback Fixes #6645.
  • Geomap: Replaced deprecated table highlight class.
  • Core: Improved handling of nested plugin/polyfill elements.
  • Site: Link to jQuery using https Fix #6658.
