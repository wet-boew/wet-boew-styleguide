Design guide

Work in progress

This page is a work in progress and requires review. Please file an issue or submit a pull request if information or coding is missing, incorrect or out of sync with the main repository (wet-boew/wet-boew).

The design guide offers a large selection of design and layout features for both basic and complex designs:

  • Core design features
    These design features are pure CSS and require no added functionality to work.
  • Plug-in design features
    These design features are a mix of JavaScript and polyfills.

Plug-in design features JavaScript and polyfill design features

Name Category Description Keywords
ARCHIVED - Archiving content Other Helps users identify Web pages that have been archived online. archived
Accessibility responsibility breakdown (WCAG 2.0) Other Provides a breakdown of WCAG 2.0 accessibility responsibilities throughout the Web production chain. arb-rra
Calendar of Events Plugin Dynamically generates a calendar interface for navigating a list of events. cal-events
Country Content Plugin A basic AjaxLoader wrapper that inserts AJAXed-in content based on a visitor's country as determined by http://freegeoip.net country-content
Data Ajax Plugin A basic AjaxLoader wrapper that inserts AJAXed-in content. data-ajax
Data Inview: Bar Plugin Displays a bar when a section moves out of the viewport. data-inview
Data Picture Plugin Event driven port of Picturefill data-picture
Datalist polyfill (auto-complete) Polyfill Adds auto-complete functionality to specific <input> fields by dynamically displaying a list of words that match the <datalist>. datalist
Details/summary polyfill (expandable/collapsible content) Polyfill The <details> and <summary> elements allows content to be expanded and collapsed. details
Equal height Plugin A plugin for responsive equal height. equalheight
Favicon Plugin This plugin provides the ability to add and update the favicon's on a web page. favicon
Feedback form Plugin Allows users to submit feedback for a specific Web page or website. feedback
Feeds Plugin Aggregates and displays entries from one or more Web feeds. feeds
Footnotes Plugin Provides a consistent, accessible way of handling footnotes across websites. footnotes
Form validation Plugin Provides generic validation and error message handling for Web forms. formvalid
Input type="date" polyfill (date picker) Polyfill The <input type="date"> element dynamically generates a calendar interface for selecting a date in a form. datepicker
Input type="range" polyfill (slider control) Polyfill The <input type="range"> element displays a slider control bar. slider
Lightbox Plugin Helps build a photo gallery on a web page. lightbox
MathML polyfill Polyfill Emulates MathML support for browsers that do not have MathML support. mathml
Menu Plugin Provides an interactive menu bar with optional sub-menus. menu
Meter polyfill Polyfill The <meter> element displays a scalar measurement in a known range. meter
Multimedia Player - Audio-only Plugin Provides an accessible audio player for embedding audio into web pages. multimedia
Multimedia Player - Video Plugin Provides an accessible multimedia player for embedding video into web pages. multimedia
Multimedia Player - YouTube Plugin Provides an accessible multimedia player for embedding YouTube video into web pages. multimedia, youtube
Overlay Plugin A flexible, responsive overlay plugin. overlay
Prettify Plugin Provides syntax highlighting of source code snippets in an HTML page using google-code-prettify. prettify
Progress polyfill (progress bar) Polyfill The <progress> element displays the progress of a task. progress
Session Timeout Plugin Helps Web asset owners to provide session timeout and inactivity timeout functionality. session-timeout
Share widget Plugin Facilitates sharing Web content on social media platforms. share
Tabbed Interface Plugin Dynamically stacks multiple sections of content, transforming them into a tabbed interface. tabs
Tabbed Interface - Carousel Plugin Dynamically stacks multiple images and captions into a carousel (or slider) widget. tabs, carousel
Tables Plugin Integrates the DataTables plugin into WET providing searching, sorting, filtering, pagination and other advanced features for tables. tables
Text highlighting Plugin Automatically highlights certain words on a Web page. The highlighted words can be selected via the query string. texthighlight
Toggle Plugin Allows a link to toggle elements between on and off states. toggle
Twitter embedded timeline Plugin Helps with implementing Twitter embedded timelines. twitter
WET theme Other The WET theme was developed to provide an example implementation of the Base Theme. theme-wet-boew
