Correctifs

Needs translation The polyfills are JavaScript components that mimic the behaviour of HTML5 tags in browsers that do not recognized such tags. They provide backwards compatibility for HTML5 in browsers that do not support HTML5.

  • Correctif « meter » (compteur) - Émule l’élément « meter » pour les navigateurs qui ne le supportent pas. L'élément « meter » affiche une mesure scalaire dans une gamme connue.
  • Correctif « progress » (barred de progression) - Émule l’élément « progress » pour les navigateurs qui ne le supportent pas. L'élément « progress » de HTML5 affiche la progression d'une tâche.
  • Correctif « datalist » (saisie automatique) - Émule l'attribut « list » de l'élément « input », ainsi que l'élément « datalist », pour les navigateurs qui ne les supportent pas. Ceci ajoute des fonctionnalités de saisie semi-automatique des champs de « input » spécifiques en affichant dynamiquement une liste de mots qui correspondent à la « datalist ».
  • Correctif « details/summary » (contenu affichable/masquable) - Émule les éléments « details » et « summary » pour les navigateurs qui ne le supportent pas. Les éléments « details » et « summary » éléments permettent au contenu d'être affiché et masqué.
  • Correctif « input type="date" » (sélecteur de date) - Émule l’élément <input type="date"> pour les navigateurs qui ne le supportent pas. Cela génère dynamiquement une interface de calendrier pour sélectionner une date dans un formulaire.
  • Correctif « input type="range" » (barre coulissante) - Émule l’élément <input type="range"> pour les navigateurs qui ne le supportent pas. Le <input type="range"> permet d'afficher une barre coulissante.
  • Correctif MathML - Émule le MathML pour les navigateurs qui ne le supportent pas.
